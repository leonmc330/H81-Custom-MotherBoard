================================================================================
Three files in this project carry code derived from the OpenBoardView project
(https://github.com/OpenBoardView/OpenBoardView):

  1. xzzpcb_parser.py — Python port of:
        src/openboardview/FileFormats/XZZPCBFile.cpp
        src/openboardview/FileFormats/XZZPCBFile.h
     (XZZPCB V1.0 parser logic and record schema, in full).

  2. rc6_native.c — C port of:
        src/openboardview/FileFormats/FZFile.cpp::FZFile::decode
     (RC6-CFB-1 cipher used to decrypt ASUS .fz files; the C fast path
     compiled into rc6_native.dll).

  3. fz_parser.py — Python port of the same RC6-CFB-1 cipher pieces from
     FZFile.cpp (specifically the `_RC6_PARITY` table, `_validate_fz_key`,
     and `_rc6_decode`). This is the pure-Python fallback used when
     rc6_native.dll is unavailable. The surrounding parser in fz_parser.py
     (Allegro Extracta record decoder, BoardModel mapping, decrypted-text
     cache, keyfile resolver) is independent original work and is licensed
     under LGPL-3.0-or-later; only the cipher pieces are MIT-derived.

The XZZ .pcb format reversal and the original C++ implementation are credited
in the OpenBoardView source as the work of:
